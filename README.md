# Event Countdown

Responsive event countdown page built with HTML, CSS, and vanilla JavaScript.

## Features
- Live countdown (days/hours/minutes/seconds) with tick animation.
- Hero/card layout with centered content and subtle grid/glow background.
- Responsive grid that collapses neatly on mobile.
- Status message flips to "Event started" at zero.

## Getting started
1. Open index.html in your browser (no build steps required).
2. Adjust the event date in script.js by editing the `targetDate` value.

## Files
- index.html — structure and copy.
- style.css — layout, gradients, and animations.
- script.js — countdown logic and date formatting.

## Customization
- Change colors or gradients in style.css under the `:root` variables and body background.
- Update CTA/link targets in index.html to your real calendar or details page.

## Notes
- Uses the Space Grotesk font from Google Fonts.
- All assets are local; works offline after the first font load.
